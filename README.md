# Prostate cancer single nuclei RNAseq and spatial transcriptomics preprocessing and analysis

## Overview
Select analysis scripts used to preprocess and analyze single-nuclei RNAseq and Xenium spatial transcriptomics data for the manuscript "Inflammatory signaling localizes to distinct tumor and epithelial cell niches in prostate tumors from men of African ancestry."

This was developed as part of my PhD dissertation work in Frankin Huang's lab at UCSF.


1. **Preprocessing** — Alignment, denoising, and integration of single-nuclei data and spatial transcriptmics data
2. **Analysis** — Cell state identification, differential expression, and cell-cell communication analysis
